# motion-sensor-light

## Goal
Build a motion-detecting light using an Arduino and PIR sensor.

## Components
- Arduino Uno
- PIR motion sensor
- LED
- Resistor
- Breadboard + wires

## How it works
When motion is detected, the sensor sends a signal to the Arduino, which turns on the LED. When no motion is detected, the LED turns off.
